Dropdown Multiselect

Add the dropdown--multiselect modifier to dropdown. This dropdown and dropdown menu will always have 100% width. Selected items are shown as "pills" below the dropdown.

Menu Filter

These are dropdown menus with filter functionality using the "typeahead" technique. While modifiers are not required, we introduce some new elements: dropdown__menu__container and dropdown__menu__scroll.
